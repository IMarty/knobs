The knobs on this page are using an old version of the knobs jQuery plugin. The new version "Knob.js" doesn't depend on jQuery and allows for many different display options like CSS, SVG, and <canvas>

Virtual knobs that work on desktops and touch based devices through the magic of javascript and HTML.
This project tries to emulate the behavior of the knobs in Apple's GarageBand for the iPad.

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/jherrm/knobs

get the source code on GitHub : jherrm/knobs
